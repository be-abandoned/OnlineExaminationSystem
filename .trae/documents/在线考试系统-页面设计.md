# 在线考试系统（网页端）页面设计说明（Desktop-first）

## 全局设计规范
- Layout：桌面端优先，内容区最大宽度 1200px；页面主体采用 12 列 Grid + 卡片式信息块；表格/列表使用粘性表头。
- Breakpoints：≥1200（桌面）；768–1199（小桌面/平板横向）；<768（移动端简化为单列，侧边栏折叠）。
- Global Styles（Design Tokens）
  - 背景：#F6F7FB；卡片：#FFFFFF；分割线：#E6E8F0
  - 主色：#2563EB；成功：#16A34A；警告：#F59E0B；危险：#DC2626
  - 字体：系统字体栈；H1 24/32，H2 18/26，Body 14/22；数字与计时可用等宽字体
  - 按钮：Primary（主色填充）、Secondary（描边）、Danger；Hover 提升阴影 + 颜色加深；Disabled 降低不透明度
  - 链接：主色下划线 hover；表格操作区用 icon + tooltip
- 通用组件
  - TopNav：左侧产品名/组织名；中部面包屑（可选）；右侧用户菜单（角色、退出）
  - SideNav：教师/学生不同菜单；在 <768 变为 Drawer
  - Toast/Modal：用于交卷确认、发布成绩确认、撤回确认

## 页面 1：登录/注册页
- Meta
  - title：在线考试系统 - 登录
  - description：账号登录后进入考试工作台
  - og:title/og:description：同上
- Page Structure：居中单卡片 + 右侧帮助信息（桌面）；移动端仅卡片
- Sections & Components
  1. 顶部：Logo + 产品名
  2. 表单卡片
     - Tabs：登录 / 注册（若你希望仅组织开通，可隐藏注册）
     - 登录：邮箱/手机号 + 密码；“忘记密码”链接；登录按钮
     - 注册：邮箱/手机号、密码、姓名；角色选择（学生/教师）或“注册后选择”
  3. 辅助区：隐私/服务条款链接；常见问题入口
- 交互状态
  - 表单校验：必填、格式错误提示；提交中按钮 loading

## 页面 2：工作台（按角色展示）
- Meta
  - title：工作台 - 在线考试系统
  - description：查看考试与成绩概览
- Layout
  - 桌面端：左侧 SideNav（240px）+ 右侧内容区（Grid 卡片 + 表格）
  - 内容区顶部：页面标题 + 关键操作按钮

### 2A 学生工作台
- Sections & Components
  1. 概览卡：待考数量 / 已交卷 / 已出分
  2. 筛选条：班级/课程（若有）、状态（待考/进行中/已交卷/已出分）、搜索
  3. 考试表格
     - 列：考试名称、开始时间、截止时间、时长、状态、操作（进入/继续/查看成绩）
     - 行状态：即将开始/进行中/已结束（用 tag 颜色区分）

### 2B 教师工作台
- Sections & Components
  1. 顶部操作：新建考试（Primary）
  2. 统计卡：已发布考试数 / 进行中 / 待阅卷（可用“待阅卷份数”）
  3. 考试列表表格
     - 列：标题、状态、时间窗、参与人数、已提交、操作（编辑/阅卷/发布状态切换）

## 页面 3：考试作答页
- Meta
  - title：考试作答 - {考试名称}
  - description：在线答题与交卷
- Layout（桌面端三栏）
  - 顶部粘性：考试信息条（名称、剩余时间、保存状态、交卷按钮）
  - 左侧：题号导航（可折叠）
  - 中间：题目内容与作答区（主列）
  - 右侧：当前题信息（分值、标记、上一题/下一题）或在小屏隐藏
- Sections & Components
  1. 考试信息条
     - 倒计时（红色临界提醒）；网络/保存状态（已保存/保存中/离线）
  2. 题目渲染
     - 支持：单选/多选/判断（Radio/Checkbox）；填空（Input）；简答（Textarea）
     - 图片/附件（若启用）：点击放大预览
  3. 交卷流程
     - Modal：未答题统计提示；确认交卷；交卷后进入成绩页或“已交卷”状态页

---

## 页面 4：考试编辑页（教师）
- Meta
  - title：考试编辑 - {考试名称/新建}
  - description：配置考试规则与组卷
- Layout
  - 顶部：标题 + 保存草稿/发布/撤回按钮
  - 主体：Tabs（基本信息 / 组卷与题目 / 分配与发布）
- Sections & Components
  1. 基本信息 Tab
     - 表单：标题、说明、时长、开始/截止时间、次数限制、题目乱序开关
  2. 组卷与题目 Tab
     - 左：题库选择（搜索、按题型筛选、批量添加）
     - 右：试卷题目列表（拖拽排序/上下移动、设置单题分值、移除）
     - 预览：弹窗或右侧抽屉展示“学生视角”
  3. 分配与发布 Tab
     - 选择班级（多选）；显示已选学生人数（由 enrollments 统计）
     - 发布前检查：时间窗、题量、总分

---

## 页面 5：成绩与阅卷页
- Meta
  - title：成绩 - {考试名称}
  - description：阅卷、发布成绩或查看本人得分
- Layout
  - 教师：左侧答卷列表 + 右侧阅卷面板（双栏）
  - 学生：单列明细（题目列表 + 得分）

### 5A 教师：阅卷与发布
- Sections & Components
  1. 答卷列表（表格）
     - 列：学生、提交时间、状态（待阅/已阅）、总分、操作（进入阅卷）
  2. 阅卷面板
     - 顶部：当前学生信息 + 保存/下一份
     - 逐题区域：显示学生答案；客观题展示自动得分；主观题输入 manual_score 与评语
  3. 成绩发布
     - 发布按钮 + 二次确认；发布后学生可见评分细节（受权限策略控制）

### 5B 学生：成绩查看
- Sections & Components
  1. 成绩头部卡：总分、排名（如不做可不展示）、出分时间
  2. 明细列表：每题题干摘要、本人答案、得分；未发布时仅显示“待出分”
